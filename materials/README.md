# Merged Source Materials

This folder is the copied-in working set from the original three project parts.

- `project_part1/`: data pipeline, chunking, OCR comparison, processed data, benchmark corpus
- `project_part2/`: retrieval notebook, top-k CSV results, cached embeddings
- `Project_part_3/`: generation/evaluation notebook, summary CSV, plots

The goal is to make `academic-rag-studio/` self-contained so future work can happen inside this one project directory instead of jumping between separate folders.

For now, the original folders still remain in the workspace unchanged.
